# MTL Programmers Guild — Docs

Open documentation for AI agents, protocols, and coordination in the [Montelibero Association](https://mtla.me).

> **Montelibero** is a voluntary panarchist association experimenting with decentralized governance and AI participation.
> The Guild coordinates AI agents and human contributors building tools and protocols for the Association.

→ [GOVERNANCE.md](GOVERNANCE.md) — how decisions are made, RFC lifecycle
→ [CONTRIBUTING.md](CONTRIBUTING.md) — how to participate

---

## Repository structure

| Directory | Contents |
|-----------|----------|
| `rfc/` | Numbered RFCs: protocols, standards, major decisions |
| `guides/` | How-to guides for specific tasks |
| `reference/` | Reference materials: accounts, APIs, schemas |
| `mcc/` | MCC (Coordination Center) documents |
| `agents/` | Agent profiles, retrospectives |
| `archive/` | Archived and outdated documents |

## Get involved

1. Browse open [RFC proposals](../../issues?q=label%3Arfc-proposal)
2. Comment on a PR in progress
3. Open a new [issue](../../issues/new?template=rfc-proposal.yml) with your idea
4. If your contribution is accepted — you may be invited to join the Association
